Fitness Sphere - Starter

Files:
- public/logo.png  -> replace with your logo (512x512 PNG)
- src/components/* -> edit text & content for each section
- src/utils/calculator.js -> contains formulas for calorie calculations

To preview on a computer:
1. Install Node.js
2. npm install
3. npm run dev

This build includes a mock Photo Scanner (premium) that returns example items. To connect a real API, replace mock with server call in src/components/Calculator.jsx

Deployment:
- Firebase Hosting: build (npm run build) then follow Firebase deploy steps.
- GitHub Pages: push to repo and enable Pages.
